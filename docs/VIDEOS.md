# Vidéos d'explication TapTapGo

## Contenu disponible par défaut

Quand l'utilisateur clique sur **« Gade Videyo Eksplikasyon »** dans Sant Èd :

- **Si aucune URL vidéo n'est configurée** : Un **diaporama visuel interactif** s'affiche, présentant les 4 étapes « Kòman Sa Mache » avec des visuels (étapes, icônes, descriptions). Les utilisateurs peuvent naviguer avec les flèches ‹ › et les points.
- **Si une URL vidéo est configurée** : La vidéo s'affiche (YouTube, Vimeo ou fichier .mp4).

## Ajouter votre propre vidéo

1. Connectez-vous en **SuperAdmin** → **Gere Sit** (Landing)
2. Remplissez le champ **« Videyo eksplikasyon — lyen »**
3. Formats acceptés :
   - **YouTube** : `https://www.youtube.com/watch?v=VIDEO_ID` ou `https://youtu.be/VIDEO_ID`
   - **Vimeo** : `https://vimeo.com/VIDEO_ID`
   - **Fichier direct** : URL vers un fichier `.mp4`, `.webm` ou `.ogg`
4. Cliquez **Anregistre**

## Créer une vidéo personnalisée

Pour une vidéo de démonstration TapTapGo :

1. **Enregistrement d'écran** : Outil type OBS Studio, Loom, ou enregistrement natif iOS/Android
2. **Montage** : CapCut, DaVinci Resolve, ou tout logiciel de montage
3. **Hébergement** : YouTube (gratuit, lien public ou non listé), Vimeo, ou hébergement sur votre serveur
4. **Sous-titres** : Recommandés en créole haïtien pour l’accessibilité
